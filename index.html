Open WebUI Docker compose.yml Creator

Our configurator generates a Docker Compose file that you can use to set up and operate Open WebUI.

Quick Guide

  1. Install Docker and Docker Compose on your device. We recommend using Docker Desktop.
  2. Configure your Open WebUI server using the options below.
  3. Create a folder on your device and save the generated compose.yml file in it.
  4. Open a terminal, navigate to that folder, and run docker compose up -d to start the server.
  5. Open your browser and connect to http://localhost:3000 (or your chosen port) to use Open WebUI.
